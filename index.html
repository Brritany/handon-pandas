Challenge 1: Create a DataFrame from a Dictionary

Scenario: Imagine you are a data analyst and you need to create a DataFrame to hold the names and ages of five persons.

Expected Output: DataFrame with columns 'Name' and 'Age'
Answer: import pandas as pd df = pd.DataFrame({'Name': ['Alice', 'Bob', 'Charlie'], 'Age': [25, 30, 35]})
Hint: Use pandas.DataFrame to create a DataFrame from a dictionary.
